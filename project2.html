Full stack JavaScript Application using a Map API to build a midpoint finder between two locations

  • Source Code

Designed and Develop the idea to create an application to find points of interest between two locations. By using MapQuest Web API services fetched JSON data while using Mapbox services to create interactive map and display the corresponding points of interest.

  • Fetched data from a mapping API to generated areas of interest for a potential meeting location between to starting locations
  • Created CRUD connections between our front-end services to a MongoDB database, manipulating the data in a useable format
  • Designed and created a user-friendly front-end using HTML and CSS to showcase all the necessary information
