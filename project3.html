Artificial Intelligence Pathfinding Simulation using Python

Using Python and practicing concepts learned about how AI learns, with a team of 4 , developed a reward-based path-finding solutions where an AI agent explores a world and finds a given destination while showcasing the steps and the most effective way to do so

  • Developed with a group a real time path finding algorithm using Q-learning and SARSA algorithms
  • Encouraged a developed AI agent based on a reward system to sort out an optimal path based on a given “world”
