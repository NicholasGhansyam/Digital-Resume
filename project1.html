Developed a UnityGame from the ground up

Created a tower defence game using C# within Unity engine that placed 3rd in an augural local game development competition.The game's core game mechanics were a top down 2D shooter where a player could collect resources and build particular towers to survive ever increasing waves of enemies. By utilizing AI path finding algoritms for the enemies, we create unqiue attributes for each type of mob.

  • Practiced software development skills to streamline the develop of the game
  • Developed AI path-find algorithms and game-logic for the player to interact with the world
  • Designed and created the playable world in which the player interacted with
